# Primer-parcial-desarrollo-web-
Desarrollo de aplicación web modularizada.  La aplicación integrará componentes reutilizables, validaciones de formulario y buenas prácticas de desarrollo.

# Café Express

Aplicación web modularizada desarrollada para el primer parcial.

## Tecnologías usadas
- HTML
- CSS
- JavaScript
- Fetch API
- Web Components

## Funcionalidades

- Login con validación en JavaScript
- Componentes reutilizables (header, sidebar, footer)
- Plantilla HTML usando <template>
- Carga de productos con Fetch desde JSON
- Web Component con Shadow DOM

## Nota

El login usa credenciales quemadas en el código con fines educativos.
